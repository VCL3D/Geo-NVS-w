Abstract

We present Geo‑NVS‑w, a geometry‑aware framework for high‑fidelity novel view synthesis (NVS) from unstructured, in‑the‑wild photo collections. In‑the‑wild methods built on volumetric density often blur edges or introduce semi‑transparent artifacts due to geometric ambiguity. Geo‑NVS‑w addresses this by rendering over a Signed Distance Function (SDF) backbone coupled with a sparse octree feature volume.

NeuS‑style SDF rendering tightly links color accumulation to the surface, while a new Geometry‑Preservation Loss (GPL) discourages transient masks from suppressing geometrically meaningful regions. Across IMC‑Phototourism scenes, Geo‑NVS‑w delivers competitive image quality with sharper details and achieves a 4–5× reduction in energy use compared to similar methods, making geometry‑centric NVS practical for in‑the‑wild settings.

Contributions

  • SDF‑guided NVS with octree features: An efficient, geometry‑first renderer pairing an SDF foreground with a background NeRF in a sparse octree‑based feature volume.
  • Geometry‑Preservation Loss (GPL): Penalizes the model for masking out geometrically significant regions as transient, ensuring coherent edges across viewpoints.
  • Measured efficiency: Single‑GPU implementation with mixed precision and power logging achieves competitive quality and 4–5× lower energy than in‑the‑wild baselines.

Method

Octree Feature Volume

We represent the scene using two sparse octrees: an SDF‑guided foreground within a unit sphere and a background NeRF. For a 3D point \(\mathbf{x}\in\mathbb{R}^3\) we trilinearly interpolate features from the foreground grid and decode via tiny MLPs into SDF \(s(\mathbf{x})\) and view‑dependent color \(c(\mathbf{x}, \mathbf{d})\). Outside, we query the background grid and pass features to a small background NeRF.

SDF‑Guided Rendering

Following NeuS, SDF samples along each ray are mapped to occupancies via an incidence term proportional to the ray–normal dot product. Alpha compositing yields per‑interval first‑hit probabilities; foreground colors are integrated around the SDF zero level set and composited over a background NeRF. Normals and derivatives use finite differences.

Appearance & Transients

A per‑image latent appearance code models illumination/white‑balance variation. Transient occluders are segmented with a lightweight CGNet trained unsupervisedly, akin to Cross‑Ray supervision.

Geometry‑Preservation Loss (GPL)

Transient masks can mistakenly suppress true structure near edges. We counteract this by penalizing mask activations on rays that pass through geometrically significant regions:

\[\mathcal{L}_{\mathrm{GPL}} = \mathbb{E}_{\mathbf{r}}\,[\, M(\mathbf{r}) \ \cdot \sigma(\, \lambda_{\text{grad}} E_{\text{grad}}(\mathbf{r}) + \lambda_{\text{curv}} E_{\text{curv}}(\mathbf{r})\,)\,]\,\]

where \(M(\mathbf{r})\in[0,1]\) is the predicted mask; \(E_{\text{grad}}=(\lVert \nabla s\rVert-1)^2\) and \(E_{\text{curv}}=|\Delta s|\) are ray‑wise averages; \(\sigma\) is a calibrated sigmoid. We use \(\lambda_{\mathrm{GPL}}=0.05\), \(\lambda_{\text{grad}}=10.0\), \(\lambda_{\text{curv}}=2.0\).

Objective

The full objective combines photometric, eikonal, curvature, Lipschitz, geometry preservation, and mask regularization terms:

\[\mathcal{L}= \lambda_{\mathrm{rgb}}\mathcal{L}_{\mathrm{rgb}} + \lambda_{\mathrm{eik}}\mathcal{L}_{\mathrm{eik}} + \lambda_{\mathrm{curv}}\mathcal{L}_{\mathrm{curv}} + \lambda_{\mathrm{mask}}\mathcal{L}_{\mathrm{reg}} + \lambda_{\mathrm{GPL}}\mathcal{L}_{\mathrm{GPL}} + \lambda_{\mathrm{lps}}\mathcal{L}_{\mathrm{lps}}.\]

During warm‑up, we additionally use a sphere‑initialization loss \(\mathcal{L}_{\text{sphere}}\).

Beyond quality, Geo‑NVS‑w is markedly efficient: a 300k‑iteration run uses about 2.05 kWh, vs. 9.7 kWh for NeRF‑W and 7.71 kWh for Ha‑NeRF, with smaller MLPs and improved surface coherence.

Conclusion

Geo‑NVS‑w advances in‑the‑wild NVS by anchoring rendering to an SDF and protecting geometry via GPL. The result is sharper, more consistent views with substantially lower energy than prior in‑the‑wild systems. Future work will explore integrating 3D Gaussian splatting with SDF priors to further cut latency while maintaining surface coherence.
